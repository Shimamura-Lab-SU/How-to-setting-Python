# クラスについて

クラスは関数や変数を内包できる**オブジェクト**です．
特に，以下の場合には関数ではなくクラスを使うと便利です．
 - 関数に初期値を設定したいとき
 - 内部変数がほしいとき

## 覚えておきたい単語

クラスには特有のワードがあるので，覚えておきましょう．

### コンストラクタ

初期化関数のこと．`__init__`で定義する．コンストラクタ内部でクラスの内部変数を定義する．
